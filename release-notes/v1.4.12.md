## v1.4.12

### 更新亮点

- 新增 TikTok 运营助手「商品管理导出」，可按店铺区域、商品状态、商品标题/ID/SKU、类目、库存和价格条件导出商品管理列表。
- TikTok 商品管理导出会拆出商品、SKU、价格、库存、近 28 天表现、其他店铺、类目、品牌、促销、违规与发品质量等字段。
- TikTok 达人视频下载结果按视频归因成交件数降序整理，并把成交件数写入计划文件名，方便优先核对高成交素材。
- TikTok 达人视频日期范围和导出文件命名继续补强，统计日期、发布时间、区域和筛选摘要更明确。
- SHEIN 商品评价精确筛选收紧为一次只选一种 SKC / 评论 ID / 星级条件，避免组合筛选导致结果口径不清。
- 补充 TikTok 商品管理、达人视频排序、入口 URL 选择、文件名上下文和 SHEIN 筛选约束回归测试。
