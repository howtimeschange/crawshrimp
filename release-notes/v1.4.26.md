## v1.4.26

### 更新亮点

- 修复森马云盘「天猫素材批量爬取-搭配购」在 Windows 后处理阶段偶发 `[WinError 2] 系统找不到指定的文件` 的问题。
- 搭配购素材包收尾阶段改为复制下载源文件进入 ZIP 打包目录，再统一清理临时文件，避免同一素材被多条对应 ID 记录复用时被提前移动。
- 保持导出行为不变：最终导出目录仍只保留搭配购素材 ZIP 和结果 Excel，不额外生成同名素材文件夹。
- 补充重复本地下载源路径的回归测试，覆盖同一个下载文件被两条搭配购记录复用的场景。
