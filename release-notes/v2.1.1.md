# 抓虾 Desktop v2.1.1

发布日期：2026-07-12

`v2.1.1` 是一次面向新设备配置与跨平台桌面壳布局的修复版本，确保 1XM Key 在 AI 生图、AI 测图和设置状态中使用同一份配置，并恢复 Windows 侧栏的正常视觉对齐。

## 修复：AI 生图与 AI 测图 Key 配置

- AI 生图工作台从设置页返回时会重新读取模型配置，避免被 `KeepAlive` 缓存的页面继续提示「未配置」。
- 设置页「AI 能力」菜单状态改为任一 1XM 图片模型 Key 已配置即显示「已配」，与实际可用模型能力一致。
- 天猫 AI 测图脚本增加回归测试，确保桌面设置写入的 1XM 4K Key 可被脚本读取并用于 `gpt-image-2` 生图。

## 修复：Windows 标题栏与侧栏布局

- macOS 的 88px 红绿灯控制区仅在 macOS 渲染环境生效，不再挤压 Windows 的 Logo 与标题栏内容。
- Windows 收起侧栏后，品牌/展开按钮保持在 56px 图标轨道内居中；macOS 保持原有安全区和收起定位。
