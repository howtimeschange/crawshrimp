## v1.4.23

### 更新亮点

- 修复 Windows 上深绘上新图包整理时 PDF 吊牌/洗唛截图失败的问题，避免 PyMuPDF 中间渲染路径过长时误报 `cannot open file`。
- PyMuPDF 渲染改为优先从内存流打开 PDF，并将中间页图命名为 `p1.png`、`p2.png` 等短文件名，降低中文路径和长文件名影响。
- 深绘上新图包与 PDF 批量截图任务改用更短的 `_pdf_work` 临时目录，最终输出图包和 `yq(1)` / `yq(2)` 命名规则保持不变。
- 补充深绘 PDF 渲染短文件名、整理图包短临时目录相关回归测试。
