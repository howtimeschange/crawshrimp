# 抓虾 Desktop v2.0.0

发布日期：2026-07-10

`v2.0.0` 是抓虾的主版本升级。产品从以 Adapter 为中心的网页自动化桌面底座，扩展为本地自动化、AI 素材生产、Prompt 管理、云端审批和受控任务机协作的一体化平台。

## 新增：AI 生图工作台

- 桌面侧边栏新增一级入口「AI 生图」，不再要求从某个 Adapter 任务进入生图流程。
- 支持单条 Prompt 与批量 Prompt；批量模式可为每条 Prompt 独立配置出图数量。
- 统一 GPT Image 与 Gemini 图像模型参数，比例变化会自动过滤可用尺寸并保持参数一致。
- 持久化 1XM 异步任务的 `task_id`、`poll_url`、轮询间隔和运行状态，应用重启后可以继续刷新。
- 生成结果写入本地缓存；远端预览 URL 异常或缓存首次失败时会进行恢复与重试。
- 新增生成结果谱系：保留来源图、编辑分支、父结果和当前图片的编辑链。
- 新增 Tldraw 标注层与二次编辑，修复标注漂移、共享任务状态串线和编辑结果归属错误。
- 任务历史支持置顶、删除、稳定排序和缓存安全清理；结果卡片、折叠区和加载状态完成响应式加固。

## 新增：本地优先 Prompt 库

- 桌面侧边栏新增一级入口「提示词库」，提供表格化分组与字段维护。
- AI 生图单次/批量入口与天猫 AI 测图任务共用 Prompt 选择器。
- 云端连接可用时支持读取云端 Prompt 库；云端草稿也可在授权的本地流程中使用。
- 云端延迟或不可用时优先保留本地可用性，避免阻塞普通生图任务。
- 增加 stale-response guard、滚动容器和请求隔离，防止切换 Prompt 库时展示旧结果。

## 新增：云端审批工作台

- 新增 `cloud/approval-workbench`，技术栈为 Cloudflare Workers、Vue 3、D1 与 R2。
- 提供管理员创建账号、登录、RBAC、角色分配和审计日志；不开放公众注册。
- 提供批次列表、单款审核、批量通过/舍弃、素材资源选择、重复资源合并和提交计划。
- 支持云端直接调用 1XM 生图、轮询请求、将完成图片写入 R2，并加入当前款式审核资源。
- Prompt 库支持 Excel 导入/导出、草稿、发布版本和按品类/性别解析模板。
- 测图数据看板支持原始工作簿导入、定时抓取、同日素材去重、跨日期累计快照和素材明细钻取。

## 新增：桌面与云端任务机协作

- 设置页新增云端审批配置、任务机注册、启动/停止与状态显示。
- 开发环境可发现本地 `127.0.0.1:8787`，生产环境使用正式审批地址，也支持显式覆盖。
- 云端审批页可以嵌入抓虾桌面端，并通过 `batch_uid` 打开指定审核批次。
- 本地批次可同步元数据与图片到云端，上传资源受批次和来源任务机作用域保护。
- 任务机使用一次性注册 token 换取长期 machine token，按 capability 领取任务。
- 派发任务支持租约、心跳、取消、幂等、失败状态和重新领取保护。
- 已实现 `generate_ai_image`、`regenerate_ai_image`、`submit_tmall_material_test`、`crawl_tmall_material_test_data` 等能力边界。

## 重构：巴拉-AI测图全链路

- 工作流可选择本地 Excel Prompt 库或线上 Prompt 库。
- 按品类、性别、字段优先级选择 Prompt，并在提交确认页选择单 Prompt 出图张数。
- 支持同款目录参考图选择、主图/平铺图组合、全盘兜底和可读的素材来源确认。
- 1XM 异步任务结果持久化，支持 504、连接中断和未完整响应等瞬时失败重试。
- AI 结果缓存到本地并在 Electron UI 中直接渲染，生成证据工作簿保留任务 ID、轮询地址、远端 URL 与本地文件。
- 支持审批后创建和直接创建两种模式；审批通知携带真实批次链接。
- 审批通过后由本地任务机下载云端资产、重建提交批次并执行天猫素材上传与测图创建。
- 测图数据导出可按任务/商品表格抓取，并可选同步到云端数据看板。

## 可靠性、安全与平台兼容

- 本地 API 继续默认要求 `X-Crawshrimp-Token`。
- 云端资源 key、文件名和同步元数据会移除本地绝对路径与敏感信息。
- 加固任务机 capability、资源下载、批次状态转换、Prompt 更新与管理员角色边界。
- 修复取消任务被重复领取、过期 Prompt 覆盖、生成任务元数据丢失、共享缓存串线等问题。
- Windows 启动时可从失效/不可写的数据目录恢复到可用目录，并统一主进程与后端的数据根目录。
- 云端审批地址在传输失败后会使缓存失效并重新探测，设置页会显示最终解析地址。

## 兼容性说明

- 主版本升级用于标记产品形态从单机自动化扩展为本地与云端协作；现有 Adapter 的 `manifest.yaml` 和 JS 注入契约继续兼容。
- 云端审批台是可选模块；未配置时不会显示云端审批入口，普通本地 Adapter 和 AI 生图仍可独立运行。
- 应用内自动更新保持关闭，请从 GitHub Releases 下载对应平台的新安装包。
