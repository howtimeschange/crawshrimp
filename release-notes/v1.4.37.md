## v1.4.37

### 更新亮点

- 桌面任务中心升级为实例化工作台，新增任务实例、草稿保存、定时计划、运行历史和产物抽屉，AI 生图审批批次可在桌面端统一管理。
- 天猫运营助手完善 AI 生图全链路：支持审批模式和直接创建模式，保留 1XM 生成证据、材料测图详情链接和历史运行复用能力。
- 天猫包装图上传继续加固，适配新版详情编辑器，跳过不再需要的旧版移动详情同步，并保持 OCR/视觉锚点校验。
- 新增 MOP 云盘素材下载、唯品会轻供款商品报表、微盟商品上新自动化等运营任务，并补充对应 manifest/auth/test 覆盖。
- 桌面后端 readiness、心跳、Electron preload/dev bridge 和任务实例 API 继续加固，减少端口占用、后台刷新和实例切换带来的中断。
