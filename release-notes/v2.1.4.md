# 抓虾 Desktop v2.1.4

发布日期：2026-07-13

`v2.1.4` 是一次面向电商自动化稳定性与 Windows 运行时恢复的补丁发布。

## Temu 评价分页

- 当 Temu 接口把请求的 20 条评价静默限制为 10 条、且响应没有返回总数或 `has_more` 分页信息时，抓取器会使用已观察到的实际页大小继续获取后续页面。
- 保留显式总数和分页标志的优先级，避免对正常接口响应改变原有结束条件。

## 天猫 AI 测图素材匹配

- 默认云盘搜索目录提升到巴拉产品上新根目录，可覆盖各季资料；仍可在任务参数中填写更深目录以限定范围。
- 主图候选排序优先 `yz(1)` 和 `yz1` 原图，再考虑橱窗海报，减少 AI 测图引用错误主素材的风险。

## Windows 运行时数据目录

- 旧 `.crawshrimp` 数据目录的标记探测遇到权限错误或其他文件系统异常时，会记录警告并继续使用平台默认数据目录。
- 显式配置的数据目录及其既有 fail-closed 语义保持不变。

## 验证

- `node --test tests/temu-single-product-reviews.test.js`
- `PYTHONPATH=. python -m unittest tests.test_runtime_paths tests.test_tmall_ai_image_chain_script tests.test_tmall_ops_manifest -v`
- GitHub Actions 将在 `main` 推送和 `v2.1.4` tag 推送后分别执行完整测试与桌面构建；tag 构建会发布签名、公证后的版本化 Release、`desktop-latest` 与 Cloudflare R2 更新镜像。
