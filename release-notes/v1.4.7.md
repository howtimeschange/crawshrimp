## v1.4.7

### 更新亮点

- 深绘上新图包整理支持自动识别洗唛 / 吊牌 PDF，按保存的截图框模板生成 yq 图片并写入同款号图包。
- 深绘 `PDF 批量截图` 的截图框模板可复用到上新图包整理任务，减少重复配置。
- 森马 / 深绘打包任务会把最终 ZIP 与结果表保留在用户可见输出目录，并清理运行时中间产物。
- 后端启动时会清理已被标记为孤儿运行态任务的运行时产物，避免历史任务残留影响后续运行。
- 适配器 SDK 文档、manifest schema 和模板已刷新，覆盖 `file_pdf`、模板下载、多 sheet 输出、当前标签页匹配和 link 开发安装。

### 产物命名

- macOS Apple Silicon：`crawshrimp-v1.4.7-mac-arm64.dmg` / `crawshrimp-v1.4.7-mac-arm64.zip`
- macOS Intel：`crawshrimp-v1.4.7-mac-x64.dmg` / `crawshrimp-v1.4.7-mac-x64.zip`
- Windows x64：`crawshrimp-v1.4.7-win-x64.exe`
- 自动更新元数据：`latest.yml` / `latest-mac.yml` / `*.blockmap`
