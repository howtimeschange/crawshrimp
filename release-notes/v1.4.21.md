## v1.4.21

### 更新亮点

- 新增抖店运营助手混合出资活动监控与订单回放脚本，支持活动范围配置、报名状态跟踪、订单回放和结果汇总。
- 抖店混合出资任务接入结构化长任务进度，桌面端可展示阶段、当前对象、成功 / 失败 / 重试摘要。
- 新增任务运行进度 API 字段与前端进度解析工具，便于后续适配器输出更细粒度的运行状态。
- 新增 Macy's 美国儿童鞋类目调研脚本，纳入 `cross-border-research` 跨境调研适配包。
- 深绘上新图包整理增强鞋包装标签图片分类，降低鞋款图片包后续人工分拣成本。
- 补充抖店、Macy's 儿童鞋、深绘鞋包装标签和任务生命周期相关回归测试。
