## v1.4.3

### 更新亮点

- Windows 桌面端后端启动链路补强：内置 Python 依赖会完整安装到 Windows bundle，后端启动支持重试并记录启动日志，减少打包版首次启动失败。
- 调度器切换为固定 UTC+8 时区，并把知识索引与调度器初始化异常隔离，避免单个初始化失败拖垮 API 服务启动。
- 森马云盘 `批量下载图片并打包` 新增“仅保留一张款号代表图”模式，输入款号时可随机选取一张款色图并按款号命名，适合代表图交付场景。
- 新增/补强 Windows 后端启动、森马代表图筛选与打包命名回归测试。

### 产物命名

- macOS Apple Silicon：`crawshrimp-v1.4.3-mac-arm64.dmg`
- macOS Intel：`crawshrimp-v1.4.3-mac-x64.dmg`
- Windows x64：`crawshrimp-v1.4.3-win-x64.exe`
