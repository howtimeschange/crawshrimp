## v1.4.6

### 更新亮点

- 新增桌面端自动更新链路，设置页可检查新版本、下载更新、查看下载进度和更新日志，并在下载完成后重启安装。
- 顶部抓虾 Logo 旁新增更新提醒按钮，检测到新版本、更新下载中或已下载待安装时会提醒用户进入更新面板。
- macOS 构建新增 zip 自动更新产物，Windows NSIS 保留 latest.yml / blockmap 元数据；GitHub Actions 支持 `v*` tag 创建正式 Release 并触发桌面 CI。
- 修复 macOS 安装新包后仍运行旧适配包的问题：内置适配包按版本覆盖旧 copy 安装，真实 link 开发模式继续保留。
- 新增/补强自动更新状态机、顶部更新提醒、适配包版本同步与 link 模式保护回归测试。

### 产物命名

- macOS Apple Silicon：`crawshrimp-v1.4.6-mac-arm64.dmg` / `crawshrimp-v1.4.6-mac-arm64.zip`
- macOS Intel：`crawshrimp-v1.4.6-mac-x64.dmg` / `crawshrimp-v1.4.6-mac-x64.zip`
- Windows x64：`crawshrimp-v1.4.6-win-x64.exe`
- 自动更新元数据：`latest.yml` / `latest-mac.yml` / `*.blockmap`
