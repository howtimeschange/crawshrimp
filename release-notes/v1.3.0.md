## v1.3.0

### 更新亮点

- SHEIN `商品分析-商品明细` 新增页面实时探测筛选参数能力，可直接在任务面板覆盖站点、SKC/SPU、上下架状态、上架日期、新品标签、商品层次与质量等级。
- SHEIN `商品分析-商品明细` 导出字段已补齐曝光、点击、加车、支付、评论、差评率、退货与质量扣款等指标，并按流量/质量分组输出。
- SHEIN 参数探测失败时会静默回退到原有执行链路，任务面板的执行进度展示也已恢复。
- Lazada `批量创建优惠券` 补强了 VN 站 live 创建与跨站点恢复流程，切站时会先回站点主页再进入目标工具页。
- Lazada ID 站当前账号若被重定向到 `account health`，现在会提前中止 live 创建并给出明确提示，避免把异常跳转当成成功路径。
- 新增/补强 SHEIN、Lazada 与 JS 执行器回归测试，覆盖动态筛选探测、跨站恢复和重定向兜底场景。
