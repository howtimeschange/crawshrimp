## v1.4.35

### 更新亮点

- 天猫运营助手新增「天猫包装图上传」工作流，支持按表格批量读取商品链接和图片包信息，把包装图写入商品 PC 详情草稿。
- 包装图上传链路加入 OCR 辅助锚点识别，可识别固定头图、洗唛、尺码、白底黑底、卖点信息等关键图段，降低纯 HTML 文本锚点失效时的误替换风险。
- 天猫详情锚点策略升级为必须命中视觉锚点后再替换，并保留全局活动/奖项详情图，避免误删页面内已有的活动说明或奖项模块。
- TikTok 达人视频下载任务兼容代理环境，视频直连下载不再强制绕开系统/网络代理。

### 验证

- 已运行 app 前端测试：`npm test`。
- 已运行 adapter JavaScript 测试：`node --test tests/*.test.js`。
- 已运行 Python 单元测试：`PYTHONPATH=. .venv/bin/python -m unittest discover -s tests -p 'test_*.py' -v`。
