## v1.2.1

### 更新亮点

- 修复 Windows 桌面端首次安装或冷启动时偶发的 `connect ECONNREFUSED 127.0.0.1:18765` 启动报错。
- Electron 主进程新增统一的后端启动控制，访问 FastAPI 前会先等待 `/health` ready，避免前端和后端冷启动抢跑。
- Windows 后端启动等待窗口从 `10s` 提升到 `30s`，更稳地覆盖首次解压、磁盘冷缓存和安全扫描带来的额外耗时。
- 后端子进程启动失败时会直接透传 `spawn` 错误或 “ready 前提前退出” 原因，便于定位真实故障。
- 新增 Node 回归测试，覆盖后端启动并发复用和启动期错误上抛。

### 产物命名

- macOS Apple Silicon：`crawshrimp-v1.2.1-mac-arm64.dmg`
- macOS Intel：`crawshrimp-v1.2.1-mac-x64.dmg`
- Windows x64：`crawshrimp-v1.2.1-win-x64.exe`
