## v1.4.22

### 更新亮点

- 桌面端后端启动与任务列表加载链路加固，后端进程启动、适配包加载或任务生命周期接口异常时会给出更稳定的状态反馈，减少启动后任务页空白或状态不一致的问题。
- 适配包加载器增强异常隔离，单个适配包元数据或路径异常时不会影响其他可用任务继续展示和运行。
- 深绘上新图包最终打包链路增强，文件夹收尾、PDF 截图失败回退和异常结果落盘更稳，降低大批量图包整理尾段失败概率。
- 桌面构建内置 Python 依赖缓存刷新逻辑增强，命中缺失 `xlrd` 等运行依赖时会自动清理旧缓存并重新下载，避免旧缓存导致新版本启动后仍缺包。
- 补充桌面后端控制器、适配包加载、任务生命周期、深绘打包和 Python 依赖下载相关回归测试。
