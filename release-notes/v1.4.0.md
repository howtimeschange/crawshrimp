## v1.4.0

### 更新亮点

- Temu `后台-商品流量-列表` 补齐当前店铺名识别与共享上下文回填，导出文件命名会优先使用真实店铺名，减少误取导航文案的问题。
- Shopee `批量创建优惠券` 新增 `优惠券名称 / 优惠码` 自定义列，优惠码现在会校验格式、阻止重复，并兼容更稳的表头别名匹配。
- Excel 导入、Temu 导出命名与 JS 执行器补充回归测试，覆盖店铺名回填、券码校验、导入保护等场景。
- GitHub `desktop-latest` 滚动构建改为正式发布，构建成功后不再显示为 Pre-release。
