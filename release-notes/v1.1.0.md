## v1.1.0

### 更新亮点

- 新增 Temu `后台-商品评价`，支持 `全球 / 美国 / 欧区` 地区切换、快捷时间范围、自定义日期范围和全量分页导出。
- Temu 后台导出文件名统一为“店铺名 + 脚本名 + 时间范围 + 操作时间”风格，覆盖商品数据、商品评价、活动数据、店铺流量等链路。
- 修复 `商品数据` 自定义日期选择链路，优先走注入并补齐 React 组件读回校验，减少日历选择失败。
- 桌面安装包命名改为稳定 ASCII 规则，避免滚动发布里出现不可读或缺失前缀的安装包名称。
- GitHub `desktop-latest` 发布说明改为读取仓库内版本化 release notes，滚动发布页面会同步展示本版更新摘要。

### 产物命名

- macOS Apple Silicon：`crawshrimp-v1.1.0-mac-arm64.dmg`
- macOS Intel：`crawshrimp-v1.1.0-mac-x64.dmg`
- Windows x64：`crawshrimp-v1.1.0-win-x64.exe`
