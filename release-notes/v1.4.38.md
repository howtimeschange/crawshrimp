## v1.4.38

### 更新亮点

- 新增 PLM 运营助手「尺码表下载器」，可按多行款号批量进入 Balabala PLM 尺寸表单并下载指定阶段尺码表，默认阶段为「大货」。
- 尺码表下载器采用 API 优先方案，复用 PLM 自身 `/csi-requesthandler/RequestHandler` 数据链路，从款式 `DataSheets` 找到 `SizeChart`，再解析 `CurrentRevision`、`Items`、`Sizes` 和 `ApparelDimension`。
- 导出 Excel 支持「宽表 + 长表」「仅宽表」「仅长表」三种格式，并通过款号、阶段、尺码表、修订版字段区分批量和多品结果。
- 任务参数新增「导出目录」选择项，完成后会把最终 Excel 复制到用户选择的文件夹；留空时继续使用抓虾运行时默认输出目录。
- 后端增加 PLM 尺码表导出目录后处理逻辑，复制时会保留运行时原文件，并在目标目录已有同名文件时自动使用唯一文件名。
