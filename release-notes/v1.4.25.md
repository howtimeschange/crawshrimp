## v1.4.25

### 更新亮点

- 新增 `mop-ops-assistant` 适配包，面向千牛素材中心 / MOP 场景，提供「批量KOL素材转短视频」和「搜推素材」两个任务入口。
- KOL 素材转短视频任务支持读取 Excel 商品 ID 或商家编码、上传本地素材图片、按比例提交展示视频生成任务，并提供仅预检与 live 提交两种运行方式。
- 搜推素材任务支持批量发布图文素材，支持商品 ID / 商家编码二选一、标题与内容描述校验、3 到 9 张图片自动上传和裁剪比例设置。
- 两个 MOP 任务都补齐 Excel / CSV 填写模板和字段说明，并支持业务图包目录、手动多选图片、商家编码 KOC 素材包等素材来源。
- 桌面任务页新增目录递归文件清单能力，选择目录后可把素材文件列表随任务参数传入，执行前也会重新扫描，减少旧文件清单导致的漏图。
- Windows 后端启动链路加固，桌面端改为按 `core.api_server` 模块方式启动 Python API；如果默认端口被旧进程占用，会切换到可用端口继续启动。
- 补充 MOP 素材规划、商家编码解析、目录素材扫描和 Windows 后端启动相关回归测试。
