## v1.2.0

### 更新亮点

- 新增 `shein-helper` 适配包，首版支持 `商品评价` 与 `商品分析-商品明细` 两条链路，默认继承当前页面筛选并自动全量分页导出。
- SHEIN `商品分析-商品明细` 支持 `当前列表 / 仅 SKC / 仅 SPU / SKC + SPU` 维度切换，以及 `昨天 / 近7天 / 近30天 / 自定义` 统计时间覆盖；导出会按 `SKC列表 / SPU列表` 双 sheet 落 Excel。
- 导出文件名上下文新增 SHEIN 页面识别逻辑，导出名会稳定回填 `Shein`、时间范围和日期区间。
- Temu `后台-商品流量-列表` 升级到更稳的恢复链路：API 单页拉取上限提升到 `500`，补齐网络超时重试、列表真实 ready 校验和恢复后重置重试预算。
- JS 执行器分页上限提升到 `1000` 页，并在分页异常持续返回 `has_more=true` 时显式抛错，避免长任务静默截断。
- 本次同步补充了 Temu 商品流量回归测试、SHEIN 适配包 Node 回归测试，以及 SHEIN 多 sheet Excel 导出测试。
